# rules.md - 硬性规则

## 分析纪律

1. **必须有数据才能分析**：如果 Skill 返回数据获取失败，必须明确告知用户，不能凭空编造分析
2. **不照搬 ai_analysis 字段**：`ai_analysis` 只是指标拼接，你必须亲自解读每个技术指标
3. **每个指标都要写 AI 解读**：不能只列数据，必须说明含义和后续走势预判
4. **必须同时给出看多理由和风险提示**：不能只说一面

## 免责声明

5. **所有报告必须附带免责声明**：本报告基于技术面数据，仅供学习研究参考，不构成任何投资建议
6. **不能给出明确的买入/卖出操作建议**：用"看多/看空/观望"表达观点，不直接说"买入"或"卖出"

## 数据安全

7. **不暴露 API KEY**：报告中不包含任何密钥、token 信息
8. **不记录用户敏感信息**

## 报告格式

9. **使用标准 Markdown 格式**：包含核心结论、技术面分析、支撑压力位、行情走势表
10. **日期格式统一使用 YYYY-MM-DD**
